# 🎹 溱媽媽原神風物之詩琴 — 智慧彈琴 

[![Platform](https://img.shields.io/badge/platform-Windows-blue.svg)](https://www.microsoft.com/windows)
[![Python](https://img.shields.io/badge/python-3.12-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

一個專為《原神》（Genshin Impact）設計的**智慧風物之詩琴/圓號自動演奏輔助工具**。本專案支援基本的多軌 MIDI 檔案解析與播放

---

## ✨ 核心特色

### 1. 🎙️音訊轉錄琴譜 (Audio-to-MIDI)
- **多格式相容**：基於 `PyAV (FFmpeg)` 全能解碼核心，相容 MP3、WAV、M4A、AAC、FLAC、WebM 等任何主流音訊格式（即使被改了副檔名也能完美處理）。
- **輕量化本機推理**：利用微軟 `onnxruntime` 後端進行本機運算，無須安裝笨重的 TensorFlow，100% 本機執行，隱私安全無虞。
- **智慧濾波降噪**：
  - **頻率限制**：自動濾除 $125\text{Hz}$ 以下（重低音、鼓點）與 $1050\text{Hz}$ 以上（高頻沙沙聲、和弦泛音）的雜音。
  - **Skyline 主旋律提取**：自動在重疊音符中只篩選音高最高的單音（人聲旋律），過濾干擾的背景伴奏。
  - **和弦時差重組**：放寬音符合併時間差至 $20\text{ms}$，自動重組零碎音符為整齊流暢的和弦。

### 2. 🎛️ 智慧播放控制與轉調系統
- **雙軌智慧轉調**：讀取新歌時，自動計算全曲黑鍵（半音）最少的移調半音數，免去手動對照琴譜。
- **動態變調 (Dynamic Modulation)**：播放時監控未來 20 個音符，隨時在播放中即時微調音高，確保所有音符皆落在白鍵上。
- **力度模擬演奏**：依據 MIDI 音符的力度（Velocity）動態縮放按鍵時間長度，強奏清脆、弱奏輕柔，完美模擬人手觸鍵感。
- **GM 樂器分析與過濾**：自動解析多軌 MIDI 中各音軌的 GM 樂器類型（如鋼琴、貝斯、打擊樂器），自動關閉低分伴奏軌，並以星級標註推薦程度。

### 3. 🌐 社群雲端預設庫與記憶系統
- **雲端同步優化**：載入無本地設定的歌曲時，自動發起網路請求，比對檔名與 MD5 從 GitHub 下載社群最佳預設設定。
- **個人偏好記憶**：自動將每首歌的移調、速度、啟用音軌、智能變調等設定以檔名為鍵值存入本地 `config.json`，再次載入時一秒還原。

## 🖥️ 介面設計

- **懸浮面板**：無邊框、精緻暗色調配色、字體與按鈕具有微調美感、支援滑鼠拖曳移動，永遠置頂（不會被遊戲視窗遮擋）。
- **鍵盤控制**：全域熱鍵控制（預設 F8 播放/暫停、F9 下一首、F7 上一首、F12 強制停止）。
- **即時顯示**：同步顯示目前播放曲目的 BPM、播放進度、音符按鍵以及目前狀態。

## 🛠️ 開發與編譯指南

### 1. 本地開發環境配置

確保您已安裝 **Python 3.12**，接著執行以下命令安裝依賴庫：

```bash
pip install -r requirements.txt
```

*主要依賴：`PyQt5` (GUI 框架), `mido` / `pretty_midi` (MIDI 解析), `onnxruntime` (AI 推理), `av` (音訊解碼), `numpy` (數據處理), `soundfile` (音訊寫入)。*

### 2. 啟動開發版

本專案因為需要模擬鍵盤信號送入高權限的《原神》遊戲視窗，**必須以系統管理員權限執行**：

```powershell
# 雙擊或在管理員終端機下執行：
run_admin.bat
```

### 3. 打包與建立安裝程式

本專案提供兩種打包方式：

#### 🟢 綠色免安裝可攜版 (PyInstaller)
將所有依賴及模型打包為一個單獨的 `.exe` 和資料夾：
```bash
python build.py
```
*編譯完成後，請至 `dist/` 目錄下取得綠色版，並以管理員身分執行裡面的 `run_admin.bat`。*

#### 📦 微軟官方安裝精靈 (MSI Installer)
基於 `cx_Freeze` 編譯並建立 `.msi` 安裝封裝檔案，**內嵌 UAC 管理員提升權限要求**（安裝後雙擊桌面捷徑會原生彈出管理員提示，無須再依賴 `.bat` 啟動）：
```powershell
# 設定 UTF-8 環境變數避免編譯路徑日文字元崩潰
$env:PYTHONIOENCODING="utf-8"
python setup_msi.py bdist_msi
```
*編譯完成後，請至 `dist/` 下雙擊 `GenshinLyrePlayer-1.0.0-win64.msi` 進行安裝。*
